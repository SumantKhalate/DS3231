◆ DS3231_Init()

HAL_StatusTypeDef DS3231_Init ( I2C_HandleTypeDef *  i2cHandle)

Initializes the DS3231 module.

Stores the i2cHandle in DS3231_device variable for further I2C communication.
Disable both the Alarm 1 (A1IE) and Alarm 2 (A2IE) interrupts
Clear both the Alarm 1 flag (A1F) and Alarm 2 flag (A2F)
Disable the battery backed square wave (BBSQW) option..

Parameters
[in]*i2cHandlePass the I2C handle pointer.
Returns
HAL_StatusTypeDef variable describing if it was successful or not.
Note
Calling this function will change the interrupt pin function (INTCN) to alarm interrupt mode.
